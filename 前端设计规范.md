## 一、技术栈

### 1.1 核心技术
- **框架**：Vue 3.4 + Composition API
- **语言**：TypeScript 5.0
- **构建工具**：Vite 5.0
- **多端方案**：UniApp (一套代码适配Web、H5、小程序、App)
- **状态管理**：Pinia 2.1
- **路由**：Vue Router 4 / UniApp内置路由
- **UI组件**：
  - Web端管理后台：Element Plus 2.4
  - UniApp多端：uView UI 3.0
- **样式方案**：SCSS + UnoCSS
- **HTTP客户端**：Axios / uni.request封装
- **图表库**：ECharts 5

### 3.2 评分星星组件

```vue
<!-- components/RatingStars.vue -->
<template>
  <view class="rating-stars">
    <view class="stars">
      <u-icon
        v-for="i in 5"
        :key="i"
        :name="i <= fullStars ? 'star-fill' : (i === fullStars + 1 && hasHalfStar ? 'star-half' : 'star')"
        :color="i <= fullStars || (i === fullStars + 1 && hasHalfStar) ? '#F59E0B' : '#D1D5DB'"
        size="16"
      />
    </view>
    <text class="rating-text">{{ rating.toFixed(1) }}</text>
    <text v-if="count" class="count-text">({{ formatCount(count) }})</text>
  </view>
</template>

<script setup lang="ts">
import { computed } from 'vue'

interface Props {
  rating: number
  count?: number
}

const props = defineProps<Props>()

const fullStars = computed(() => Math.floor(props.rating))
const hasHalfStar = computed(() => props.rating % 1 >= 0.5)

const formatCount = (count: number) => {
  if (count >= 1000) {
    return `${(count / 1000).toFixed(1)}k`
  }
  return count
}
</script>

<style lang="scss" scoped>
.rating-stars {
  display: flex;
  align-items: center;
  gap: $spacing-2;

  .stars {
    display: flex;
    gap: 2px;
  }

  .rating-text {
    font-size: $font-size-sm;
    font-weight: $font-weight-medium;
    color: $text-primary;
  }

  .count-text {
    font-size: $font-size-xs;
    color: $text-secondary;
  }
}
</style>
```

  uni.navigateTo({ url: '/pages/ai/chat' })
