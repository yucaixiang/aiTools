## 五、快速开始指南 🚀

### 方案A：立即体验基础服务（5分钟）

```bash
# 1. 进入部署目录
cd deployment

# 2. 启动所有基础服务
docker-compose up -d

# 3. 等待服务启动（约30秒）
docker-compose ps

# 4. 访问服务
# MySQL: localhost:3306 (root/root123456)
# Redis: localhost:6379 (password: redis123456)
# Elasticsearch: http://localhost:9200
# Kibana: http://localhost:5601
# Portainer: https://localhost:9443
```

### 方案B：开始后端开发（使用脚手架）

```bash
# 1. 使用Spring Initializr生成项目
curl https://start.spring.io/starter.zip \
  -d dependencies=web,data-jpa,mysql,redis,lombok \
  -d bootVersion=3.2.0 \
  -d javaVersion=17 \
  -d groupId=com.toolrecommend \
  -d artifactId=tool-service \
  -o tool-service.zip

# 2. 解压到backend目录
unzip tool-service.zip -d backend/tool-service/

# 3. 参考《技术架构文档.md》开发业务代码
```

### 方案C：开始前端开发

```bash
# 1. 创建UniApp项目
npx degit dcloudio/uni-preset-vue#vite-ts frontend/uniapp

# 2. 安装依赖
cd frontend/uniapp
npm install

# 3. 启动开发服务器
npm run dev:h5

# 4. 参考《前端设计规范.md》开发页面
```
