## 🚀 核心功能详解

### 1. 工具管理系统

**功能特性**:
- 工具CRUD（创建、查询、更新、删除）
- 多维度筛选（分类、定价、评分、标签）
- 排序功能（热门、最新、评分、评论数）
- 点赞收藏系统
- 浏览量统计
- 相似工具推荐
- 缓存优化（热门工具60分钟缓存）

**技术实现**:
- MyBatis Plus动态查询
- Redis缓存热门数据
- 定时任务更新统计数据

### 2. AI对话推荐

**功能特性**:
- 多轮对话（保持上下文）
- 智能意图识别
- 工具推荐（基于关键词匹配）
- 对话历史管理
- 会话创建和切换

**技术实现**:
- Ollama本地部署（qwen2.5:7b模型）
- OkHttp调用AI API
- 对话历史持久化到MySQL
- 关键词提取和匹配算法

### 3. 评论评分系统

**功能特性**:
- 星级评分（1-5星）
- 文字评论（500字内）
- 优缺点标签
- 有帮助投票
- 评分分布统计
- 热门评论展示
- 权限验证

**技术实现**:
- Redis记录投票状态
- JSON存储优缺点数组
- 软删除机制
- 评分聚合查询

### 4. 用户认证系统

**功能特性**:
- 用户注册（用户名/邮箱验证）
- 用户登录（账号/密码）
- Token刷新机制
- 记住我功能
- 个人信息管理

**技术实现**:
- BCrypt密码加密
- JWT Token（7天有效期）
- Refresh Token（30天有效期）
- Redis存储会话

### 5. 管理后台

**功能特性**:
- 工具审核管理
- 用户管理
- 评论审核
- 数据统计
- 操作日志

**技术实现**:
- 管理员权限验证
- 审核流程管理
- 数据统计查询
