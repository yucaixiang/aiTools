# AI工具推荐系统 - 设计文档总览

## 项目简介

**项目名称**：智能工具推荐平台
**版本**：v1.0
**创建日期**：2025-12-04
**项目定位**：基于AI的软件工具发现和推荐平台

---

## 核心特性

### 🎯 产品特色
- **AI对话推荐**：通过自然语言对话，智能推荐最适合的工具
- **分类浏览**：多级分类体系，快速定位所需工具
- **评分评论**：用户真实评价，帮助决策
- **多端适配**：Web、H5、小程序、App全覆盖
- **替代工具查找**：为现有工具寻找更优替代方案
- **使用指导申请**：提供工具部署和使用咨询

### 🚀 技术亮点
- **混合AI架构**：本地Ollama + 云端通义千问，平衡成本与质量
- **向量检索**：Qdrant向量数据库实现语义搜索
- **微服务架构**：Spring Boot 3 + K8s，高可用、可扩展
- **一套代码多端**：Vue3 + UniApp，提高开发效率
- **智能推荐算法**：多维度评分，精准匹配用户需求

## 核心功能模块

### 1. 工具展示系统
- 首页（Banner、分类、热门工具、新品推荐）
- 工具列表（分类筛选、标签筛选、高级筛选、排序）
- 工具详情（基本信息、评分、评论、替代工具）
- 搜索功能（全文检索、搜索建议、热门搜索）
- 排行榜（今日/本周/本月/新品/口碑/分类）

### 2. AI对话推荐系统
- 智能对话（意图识别、实体提取、多轮对话）
- 工具推荐（向量检索、多维度排序、推荐理由生成）
- 对话历史（查看、继续、删除、导出）
- 反馈机制（点赞/点踩、点击工具、追加评论）

### 3. 用户系统
- 注册登录（手机号、邮箱、微信、第三方）
- 个人中心（基本信息、统计数据）
- 收藏管理（收藏夹、批量操作）
- 浏览历史
- 评分评论
- 申请记录（下架、替代、指导）

### 4. 后台管理系统
- 工具管理（审核、编辑、下架）
- 分类标签管理
- 用户管理（禁用、角色管理）
- 评论审核
- 工单处理
- 数据统计
